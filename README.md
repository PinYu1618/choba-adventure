# Choba Adventure

## Keys

- **Esc**: If in `MainMenu`, quit app. If in `InGame`, back to `MainMenu`

* Only when run with feature `dev`:
  - **Enter**: If in `MainMenu`, enter `InGame`
  * **LShift + Esc**: Quit app
  * **Del**: Restart game. This will exit `InGame` then enter `InGame` again
